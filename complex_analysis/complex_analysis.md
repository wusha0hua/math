# the algebra of complex number

$$
(a + bi) + (c + di) = (a + c) + (b + d)i \\ \\

(a + bi)(c + di) = (ac - bd) + (ad + bc)i \\ \\

\frac{a + bi}{c + di} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i \\ \\

\sqrt{a + bi} = \pm (\sqrt{\frac{a + \sqrt{a^2 + b^2}}{2}} + \frac{b}{|b|}\sqrt{\frac{-a + \sqrt{a^2 + b^2}}{2}}i)
$$
